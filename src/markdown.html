STATS NTILE ANALYSIS Extension Command

Display statistics for the ntiles of a dataset of predicted problabilities and associated charts

STATS NTILE ANALYSIS
PREDPROB* = predicted probabilities variable name
DEPVAR* = dependent variable from the analysis
PREDVAL* = predicted value to which the probabilities apply
NTILES* = number of ntiles for the table

/CHARTS
GAIN = NO** or YES
RESPONSERATE = NO** or YES
LIFT = NO** or YES

/HELP

* Required
** Default

STATS NTILE ANALYSIS /HELP prints this help and does nothing else.


STATS NTILE ANALYSIS PREDPROB = PredictedProbabilities_1 DEPVAR=credit_rating NTILES=10
PREDVAL=1
/CHARTS GAIN=YES RESPONSERATE=YES LIFT=YES

Note that variable names are case sensitive in this procedure and must match the case in the dataset.

PREDPROB specifies the variable containing the predicted probabilities. Often this would come from the estimation sample, but it could, alternatively, come from another dataset now open that has been scored using the Sampling Wizard or another method.

DEPVAR is the dependent variable from the analysis that produced the predicted probabilities.

NTILES specifies the number of ntiles for the table. The default is 10.

PREDVAL specifies the value to which the predicted probabilities apply and for which analysis is desired. It can be either string or numeric.

CHARTS

GAIN specifies whether or not to produce a gains chart showing the gain percentages by ntile

RESPONSERATE specifies whether or not to produce a response rate chart showing the response rates by ntile

LIFT specifies whether or not to produce a lift chart showing the lift by ntile
